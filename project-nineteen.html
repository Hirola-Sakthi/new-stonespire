About the Project

This minimal house is planned in such a way that the building incorporates all the spaces in a three-storey building for easy access. The use of cladding material on the exterior adds a striking character to the whole residence.

This residence is defined by vertical elements broken by horizontal planes that create a rhythmic pattern across the building. A humble yet dignified structure that fulfills the client’s needs perfectly.
